◆ 作品について ◆

◇使用言語：C++
◇SDK     ：DirectX9
◇開発期間：2021/11〜2022/2(4ヶ月)
◇開発人数：1人
◇動作環境：Windows10
　　　　　　XInput対応可能のゲームパッドに対応しています。

◆ 作品の内容 ◆

ダッシュで助走をつけてからジャンプをし、踏み切りのラインから
どこまで遠くまで跳べるかを競うアクションゲームです。

スコアアタックやCOMとの対戦ができる「ひとりようモード」と、
2人で対戦ができる「ふたりようモード」で遊ぶことができます。

◆ 操作方法 (キーボード・ゲームパッド) ◆

『ダッシュ』    …「Ａキー」と「Ｄキー」を交互に押す。
　　　　　　　　　 または、左スティックの左右を交互に倒す。
                     
『ジャンプ』    …「スペースキー」または「Ａボタン」
                     
『空中ジャンプ』… ジャンプした後、「スペースキー」または「Ａボタン」を押す。
 
『ホバリング』  … 空中ジャンプの後、「スペースキー」または「Ａボタン」を押す。
                    連打すると浮き続けることができます。
 
『ポーズ』      … 「Ｐキー」または「スタートボタン」
 
『決定』        … 「スペースキー」または「Ａボタン」
                     メニュー画面などで使います。
 
『キャンセル』  … 「バックスペースキー」または「Ｘボタン」
                     メニュー画面などで使います。
 
 (〜 キーボード操作で2人目が遊ぶ場合 〜)
 ( Ａ、Ｗ、Ｓ、Ｄの代わりに、「十字キー(←↑↓→)」を使用します)
 (「スペースキー」の代わりに、「エンターキー」を使用します)
